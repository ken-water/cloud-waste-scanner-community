# Community Skills Copy Pack

This document provides reusable outward-facing copy for Community skills.

## 1. GitHub README short section

```md
### Community Skills

Cloud Waste Scanner Community now includes local-first skills that help a single operator get more value from scan evidence.

- `cws-report-explainer`: explain findings for operators, finance, and leadership
- `cws-weekly-brief`: summarize current vs prior evidence
- `cws-playbook-writer`: turn findings into safer action playbooks
- `cws-export-auditor`: review exports before sharing them

Community skills work best when connected to the CWS local API, but can also run from exported evidence files.
```

## 2. Website copy

### Headline

Local-first skills for explaining, reviewing, and preparing action from cloud waste evidence.

### Supporting copy

Cloud Waste Scanner Community is not just a scanner. It now includes a growing skill surface for turning raw findings into usable outputs: explanations for operators, summaries for finance and founders, weekly review notes, safer cleanup playbooks, and export audits before you share reports outside the product.

### Skill list

- Explain findings with `cws-report-explainer`
- Review weekly change with `cws-weekly-brief`
- Prepare safer cleanup steps with `cws-playbook-writer`
- Tighten exports with `cws-export-auditor`

### Runtime note

These skills work best when connected to the local CWS API, but they can also operate on exported evidence files.

## 3. Blog intro copy

Cloud Waste Scanner Community is moving beyond raw detection output. With the first Community skill family, users can now explain findings, compare this week against the prior run, prepare safer cleanup actions, and audit exported reports before sending them to finance or leadership. The result is a more usable local-first workflow without forcing teams into a hosted control plane.

## 4. Chinese short copy

Cloud Waste Scanner Community 现在不只有扫描能力，也开始具备本地优先的 skills 能力。用户可以直接基于本地 API 或导出文件，解释 findings、生成周报、整理更安全的执行步骤，并在对外分享前检查报告是否足够清楚。

## 5. Product rule reminder

When reusing this copy, keep the boundary explicit:

- Community skills explain and prepare
- Team skills coordinate and execute
- Enterprise skills control and audit
