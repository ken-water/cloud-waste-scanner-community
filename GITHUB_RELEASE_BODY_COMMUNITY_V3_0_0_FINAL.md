# Cloud Waste Scanner Community v3.0.0

Selected as the recommended GitHub Release page body.

Use this version for the public release page.

Keep `GITHUB_RELEASE_BODY_COMMUNITY_V3_0_0.md` as the longer reference version when more context is needed.

Community `v3.0.0` resets the release line for the open client-first roadmap.

This release clarifies the Community / Team / Enterprise boundary and introduces the first Community skill family for local evidence explanation, weekly review, action preparation, and export quality checks.

This release-line reset does **not** imply that every unchanged desktop or SDK binary artifact was rebuilt as `3.0.0`.

## Highlights

- Community release line reset to `3.0.0`

## Community skills now available

- `cws-report-explainer`: explain findings for operators, finance, and leadership
- `cws-weekly-brief`: summarize current vs prior evidence
- `cws-playbook-writer`: turn findings into safer action playbooks
- `cws-export-auditor`: audit exports before sharing

## Upgrade guidance

1. keep using the local-first desktop workflow
2. use Community skills through:
   - connected mode: installed app + local API
   - file-only mode: exported evidence
3. treat Team and Enterprise skill ideas as roadmap, not current GA surface

## Scope note

Describe this as:

- a Community release-line reset
- a Community skill-surface release
- a clarification of edition boundaries

Do not describe it as:

- a major binary rebuild
- completion of Team or Enterprise product lines
