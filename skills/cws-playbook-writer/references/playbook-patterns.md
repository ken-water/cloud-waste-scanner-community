# Playbook Patterns

## Single-finding playbook

Use:

1. What this finding probably means
2. Pre-check
3. Execution suggestion
4. Verification
5. Rollback caution

## Category playbook

Use:

1. Why this category appears
2. Low-regret candidates first
3. Common false-positive traps
4. Verification after cleanup
5. Rollback caution

## Tone

- direct
- operational
- cautious
- no generic cloud theory
