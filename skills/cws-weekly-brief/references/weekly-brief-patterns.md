# Weekly Brief Patterns

## Operator weekly brief

Use:

1. headline
2. biggest change from prior run
3. top provider or resource-type concentration
4. recurring or repeated concern
5. next review action this week

## Founder weekly brief

Use:

1. one-paragraph business summary
2. whether the waste picture improved or worsened
3. where attention is concentrated
4. what decision should happen this week

## Finance checkpoint

Use:

1. current projected monthly waste
2. direction vs prior run
3. biggest cost bucket
4. what looks stable versus still under review
