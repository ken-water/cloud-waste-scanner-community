# Output Patterns

## Operator

Use this structure:

1. Headline: what the scan found
2. Highest-confidence cleanup candidates
3. What still needs owner confirmation
4. Safe next actions this week

## Finance

Use this structure:

1. Projected monthly waste in source currency
2. What portion looks high confidence vs review-needed
3. Main cost buckets by provider or resource type
4. What can likely change before the next invoice

## Executive

Use this structure:

1. One-paragraph operating summary
2. Biggest concentration points
3. Where the process is weak: missing owners, stale review rhythm, unclear accountability
4. What decision the leader should force this week

## Weekly Action List

Prefer a flat table or bullet list with:

- priority
- resource or finding label
- reason
- expected impact
- risk level
- owner status
- next action

## Tone

- Direct
- Specific
- No generic cloud-cost theory
- No "AI consultant" phrasing
